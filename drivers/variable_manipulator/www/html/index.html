Variable tokens

            
PARAM{deviceId,variableId} reads a variable's current value. Click a
            variable in the browser to insert its token at the cursor. The variable can also be
            referenced by name: PARAM{96,HUMIDITY} (exact match, case sensitive).

          

          

            
Equation mode

            
The substituted text is evaluated as math and must produce a number.

            
Operators: + - * / % ^ ( )

            
Comparisons: == ~= < <= > >= with and,
            or, not for conditionals:
            PARAM{96,1002} > 50 and 1 or 0

          

          

            
Functions

            
abs ceil floor sqrt
            min max round(x, digits)
            c2f f2c tonumber pi
            huge

            
The full math.* and string.* libraries are available,
            for example math.random(1, 10).

          

          

            
String mode

            
Literal text with tokens substituted, no evaluation:
            Espresso: PARAM{2795,1017} W
